# Racnetul_Carpatilor_site
**Răcnetul Carpaților** este un site de știri online dedicat cititorilor care vor să fie la curent cu cele mai importante evenimente din țară și din lume, dar și cu informații utile din viața de zi cu zi.

Acest repository conține codul sursă pentru site-ul *Răcnetul Carpaților*.  
Proiectul include structura paginilor, fișierele de stil (CSS), componentele interactive și funcționalitățile de comentarii și contact.
